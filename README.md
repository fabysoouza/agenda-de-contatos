# 📖 Agenda de Contatos

Um aplicativo web simples e responsivo para gerenciamento de contatos. Este projeto foi desenvolvido para colocar em prática conceitos fundamentais de desenvolvimento front-end, incluindo manipulação do DOM e formatação de dados em tempo real.

🔗 **[Acesse o projeto online aqui](https://agenda-de-contatos-gold-xi.vercel.app/)**

---

## 🚀 Funcionalidades

* **Adição de Contatos:** Inserção de nome e telefone em uma tabela dinâmica.
* **Máscara de Telefone Automática:** O campo de telefone formata os números automaticamente enquanto o usuário digita (padrão `(DD) 99999-9999`), impedindo a entrada de letras.
* **Design Responsivo:** A interface se adapta a dispositivos móveis e desktops.
* **UI/UX:** Tabela com efeito "zebra" para facilitar a leitura e botões com efeitos interativos (hover).

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do formulário e da tabela.
* **CSS3:** Estilização com Flexbox, estilização de tabelas e Media Queries para responsividade.
* **JavaScript:** * Captura de eventos do formulário (`addEventListener`).
  * Manipulação do DOM (`innerHTML`, `value`).
  * Expressões Regulares (Regex) para a criação da máscara de telefone.
